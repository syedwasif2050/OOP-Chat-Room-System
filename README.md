### Python OOP: Chat Room System
A console-based Chat Room application developed using Object-Oriented Programming (OOP) principles in Python.

**Key OOP Concepts Demonstrated:**
- **Encapsulation & Classes:** Structured data using distinct classes for `User`, `Message`, and `ChatRoom`.
- **Object Aggregation:** The `ChatRoom` class manages lists of `User` objects and `Message` objects dynamically.
- **State Management:** Simulates real-time interactions including joining a room, leaving a room, sending text messages, and rendering full chat histories.
